# bravepeach_web
BravePeach WebServer with Django

## Settings for Developers

### Pre-requisites

```
$ sudo apt install python3-dev python3-pip git git-flow
```

### Instructions

```
$ git clone [this repo]
$ cd web
$ git flow init
[Several Enters]
$ virtualenv .venv
$ . .venv/bin/activate
(.venv)$ pip install -r requirements.txt
```
